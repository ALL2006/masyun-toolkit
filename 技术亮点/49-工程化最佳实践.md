## 💡 项目应用

### 测试配置

```python
# pytest.ini
[pytest]
testpaths = tests
python_files = test_*.py
python_classes = Test*
python_functions = test_*
addopts = --cov=app --cov-report=html --cov-report=term
```

### Docker配置

```dockerfile
# 前端：多阶段构建

# ... 部署
```
