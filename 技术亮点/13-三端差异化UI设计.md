## 📌 技术概述

### 设计理念

针对**三类不同用户群体**（游客、企业、政府），我们设计了**完全差异化的UI界面**：

| 端 | 用户类型 | 核心需求 | 设计风格 |
|---|---------|---------|---------|
| **游客端** | 个人游客 | 行程规划、景点推荐 | 亲和友好、卡片式 |
| **企业端** | 景区/酒店 | 数据分析、客户洞察 | 专业商务、图表密集 |
| **政府端** | 文旅部门 | 客流监控、决策支撑 | 指挥舱、大屏展示 |

#### 智能对话界面

```typescript
// frontend/src/pages/tourist/TouristChat.tsx

export default function TouristChat() {
  return (
    <Card
      title={
        <Space>
          <RobotOutlined />
          <span>智能导游助手</span>
        </Space>
      }
      extra={<Tag color="green">在线</Tag>}
    >
      {/* 消息列表 - 对话气泡样式 */}
      <div className="chat-messages">
        {messages.map(msg => (
          <div key={msg.id} className={`message ${msg.role}`}>
            {/* 用户消息 - 右对齐，蓝色背景 */}
            {msg.role === 'user' && (
              <div className="user-message">
                <Avatar icon={<UserOutlined />} />
                <div className="bubble">{msg.content}</div>
              </div>
            )}

            {/* AI消息 - 左对齐，白色背景 + 建议按钮 */}
            {msg.role === 'assistant' && (
              <div className="assistant-message">
                <Avatar icon={<RobotOutlined />} />
                <div className="content">
                  <div className="bubble">{msg.content}</div>

                  {/* 建议问题按钮 */}
                  {msg.suggestions && (
                    <div className="suggestions">
                      {msg.suggestions.map((suggestion, i) => (
                        <Button
                          key={i}
                          size="small"
                          type="dashed"
                          onClick={() => handleSuggestionClick(suggestion)}
                        >
                          💡 {suggestion}
                        </Button>
                      ))}
                    </div>
                  )}
                </div>
              </div>
            )}
          </div>
        ))}
      </div>

      {/* 输入框 - 圆角大按钮 */}
      <div className="chat-input">
        <Input.Search
          placeholder="问我任何关于山西旅游的问题..."
          enterButton="发送"
          size="large"
          onSearch={handleSend}
        />
      </div>
    </Card>
  );
}
```

## 🎯 设计对比总结

### 视觉设计对比

| 维度 | 游客端 | 企业端 | 政府端 |
|-----|-------|-------|-------|
| **主色调** | 紫蓝渐变 | 蓝绿商务 | 深蓝+红 |
| **字体大小** | 较大(16px) | 中等(14px) | 灵活(12-36px) |
| **圆角** | 大(8px) | 中(4px) | 小(2px) |
| **阴影** | 柔和 | 适中 | 强烈 |
| **动画** | 丰富 | 适中 | 简洁 |

### 功能布局对比

| 端 | 导航方式 | 内容密度 | 交互频率 |
|---|---------|---------|---------|
| **游客端** | 标签页 | 低 | 高 |
| **企业端** | 菜单+卡片 | 中 | 中 |
| **政府端** | 仪表板 | 高 | 低 |

### 数据展示对比

| 端 | 图表类型 | 数据粒度 | 实时性 |
|---|---------|---------|-------|
| **游客端** | 卡片、列表 | 高层次 | 实时 |
| **企业端** | 饼图、柱状图 | 中等 | 定期更新 |
| **政府端** | 热力图、排名表 | 详细 | 实时 |
